MyPortfolio
Веб-сайт-портфолио с информацией об услугах. Чистая семантическая верстка без использования JavaScript и фреймворков.
-  Чистая HTML/CSS верстка
-  Полностью адаптивный дизайн
-  Семантическая разметка
-  Без зависимостей и фреймворков
-  Быстрая загрузка страниц
-  Оптимизированные изображения
-  Доступность (ARIA-атрибуты)
  
 Структура сайта

1. **Главная секция** - Приветствие и краткое представление
2. **Обо мне** - Информация о навыках и опыте
3. **Услуги** - Перечень предоставляемых услуг
4. **Портфолио** - Примеры выполненных работ
5. **Контакты** - Способы связи со мной
